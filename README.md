# zerobase_wifi_service

# Seoul-Wisi-Service
 * 사용자 기반 서울시 공공와이파이 정보 제공 서비스

## 🎯 사용 기술 및 버전
 * JAVA (JDK 1.8), JSP (Maven)
 * JDBC (3.0.7)
 * MariaDB
 * JSON-Simple (1.1)
 * Lombok (1.18.24)
 * Tomcat (8.5.82)
 * HTML, CSS, JS
 * 서울시 공공 Wifi Open Api
   
## 🎯 구현한 기능
 * Open API 정보 가져오기
 * 내 위치 or 입력 위치를 입력으로 
 * 가까운 위치의 와이파이 20개 보여주기
 * 입력 값 위치정보와, 요청 시간을 저장하는 
	  히스토리를 저장하고 보여주는 기능 구현
